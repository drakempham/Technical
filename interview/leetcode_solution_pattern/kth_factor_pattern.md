# Factor Pairs From Both Sides || Beginner Friendly ⛳️ || Square Root Thinking

## Intuition

Factors come in pairs.

If `x` is a factor of `n`, then `n // x` is also a factor of `n`.

For example, with `n = 12`:

```text
1 x 12
2 x 6
3 x 4
```

The smaller factors are found before or at `sqrt(n)`, and the larger factors are their paired values. So instead of checking every number from `1` to `n`, we only scan up to `sqrt(n)`.

The first half gives us factors in increasing order. The second half can be generated by reversing the smaller factors and converting each `x` into `n // x`.

## Approach

- **Step 1:** Scan numbers from `1` while `num * num <= n`.
- **Step 2:** Whenever `num` divides `n`, store it as a small factor.
- **Step 3:** If `k` is within the small factors list, directly return `small_factors[k - 1]`.
- **Step 4:** Otherwise, subtract the number of small factors from `k`.
- **Step 5:** Reverse the small factors list and use each factor's pair `n // factor`.
- **Step 6:** Skip the duplicate middle factor when `n` is a perfect square.
- **Step 7:** Return the paired factor when `k` reaches `0`; otherwise return `-1`.

## Complexity

- Time complexity: `O(sqrt(n))`
- Space complexity: `O(sqrt(n))`

## Code

**Python:**

```python
class Solution:
    def kthFactor(self, n: int, k: int) -> int:
        small_factors = []
        factor = 1

        while factor * factor <= n:
            if n % factor == 0:
                small_factors.append(factor)
            factor += 1

        if k <= len(small_factors):
            return small_factors[k - 1]

        k -= len(small_factors)

        for factor in reversed(small_factors):
            paired_factor = n // factor

            if paired_factor == factor:
                continue

            k -= 1
            if k == 0:
                return paired_factor

        return -1
```
